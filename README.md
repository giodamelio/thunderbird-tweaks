# Account Color Stripe

A Thunderbird extension that draws a colored stripe on each message-list row, using the account's native color.

Installs from the latest [GitHub release](https://github.com/giodamelio/thunderbird-tweaks/releases) and auto-updates from there.

## Build locally with Nix

```sh
nix build   # -> result/account-stripe-<version>.xpi
```
